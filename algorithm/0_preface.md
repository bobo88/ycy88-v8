:::

先抛出几个问题吧！

::: info 1）数据结构的本质？
数据结构的本质是组织和存储数据，以便高效地访问和修改。
:::

::: info 2）为什么会有不同的数据结构？
不同的数据结构为各种具体问题提供了特定的组织方式和操作效率，以满足不同的性能和功能需求。
:::
