## 一、Java 咖啡真好喝

::: tip 提示
一经编写到处运行（跨平台能力）
:::

### 1）特点：

- 简单
- 面向对象
  - 封装
  - 继承
  - 多态
- 分布式
- 体系结构中立
- 可移植性
- 解释执行
- 健壮性
- 多线程

### 2）Java 平台

- Java SE
- Java EE
- Java ME

### 3）Java 虚拟机

Java 应用程序能够跨平台运行，这主要是通过 Java 虚拟机实现的。

编译运行流程：Java 源文件 -- （编译器：编译） -- 字节码文件（.class 文件） -- （解释器：解释） -- 计算机。
