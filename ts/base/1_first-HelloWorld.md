# TS 序章 - 第一个 HelloWorld

安装 Typescript：

```js
$ npm install -g typescript
```

运行上述命令后，全局安装并可以执行 `tsc` 命令，编译 TS 文件：

```js
$ tsc HelloWorld.ts
```

编写第一个 HelloWorld.ts

```js
// 定义hello函数，其参数有且只有一个，且必须是字符串类型
function hello(name: string) {
  return `你好，我的名字叫${name}`
}
hello('Bob')
```

增加函数返回值类型约束，返回 string 类型

```js
// 增加函数返回值类型约束: string
function hello(name: string): string {
  return `你好，我的名字叫${name}`
}
hello('Bob')
```

函数中有多处类型约束保持一致，可以考虑设置「泛型」

```js
// 泛型
function hello<T>(name: T): T {
  return name
}
hello < string > 'Bob'

// 拓展性更好，设置 number 也可以（类似ID）
hello < number > 123
```

::: tip tsconfig
tsconfig 文件主要用来控制 typescript 编译器(tsc, typescript compiler)的一些行为, 比如指定哪些文件需要让 tsc 来编译, 哪些文件不想让 tsc 进行编译, 设置是否去掉注释、编译成哪个版本的 js 等。
:::
全局安装 TS 后，运行 `tsc` 可以发现有很多配置相关的命令，比如可以通过 `tsc HelloWorld.ts --removeComments` 将注释去掉。

生成 tsconfig.json 文件的命令：

```js
// 生成tsconfig.json文件
$ tsc --init
```
