# 经济学原理

十几年前，我就在大学里接触学习了经济学，但当时对它的理解还停留在表面。这些年来，断断续续的阅读和思考，依然还是没有形成系统性的认识。

最近，我决定重新审视经济学，并尝试用一种新的方式来理解它、学习它、应用它，也分享给更多的人。

- 从 **个人层面** 来说，学好经济学，让你更聪明地生活，做更好的决策，理解财富管理，预测趋势，避免风险。
- 从 **社会层面** 来说，学好经济学，让你更清晰的认识社会运作的底层逻辑，理解经济现象，预测政策影响，推动社会进步。
- 从 **哲学层面** 来说，学好经济学，让你拥有更全面的思维框架，训练理性与批判思维，理解世界的复杂性，提升自我认知。

正如诺贝尔经济学奖得主保罗·萨缪尔森所说：“**经济学是一门关于稀缺资源最优分配的科学，但它的价值，远远超出了数字的范畴。**”

这篇文章只是万丈高楼起的伊始，希望你能和我一起，继续探索经济学，一起学习，一起成长。

我简单罗列一些常见的社会问题，后续我将逐一思考审视这些以及更多其他问题，希望你我都能从经济学原理中找到答案。
