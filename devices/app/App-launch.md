# App 上架流程

众所周知，**应用市场**主要分为两类，一类是 **App Store**，另一类是**安卓市场**。​

而安卓市场又分为：**第三方市场**（如：应用宝、360 手机助手、豌豆荚），和**手机厂商市场**（如：华为、OPPO、VIVO、小米等）。​

如果想在这些主流应用市场上架我们的 APP，那么上架之前都必须要准备的一项就是：**计算机软件著作权**。​

计算机软件著作权，是指软件开发者或者其他权利人依据有关著作权法律的规定，对于软件作品所享有的各项专有权利。就权利的性质而言，它属于一种民事权利，具备民事权利的共同特征。​

著作权是知识产权中的例外，因为著作权的取得无须经过个别确认，这就是人们常说的“自动保护”原则。软件经过登记后，软件著作权人享有发表权、开发者身份权、使用权、使用许可权和获得报酬权。​

因为咱们 APP 要上线应用市场，需要证明这款 APP 是咱们自己，怎么证明是我们自己的呢？就需要我们对 APP 申请计算机软件著作权。​

有了软件著作权，除了在发生争议时可以更好的保障自己的权益，从商业角度而言也会让自己的软件更有价值。如果软件有了贸易版权，那么拥有软件著作权的软件价值还将翻倍。​

## 一、安卓

> 在上架前，您需要准备好相关的资料，包括 APK 包、图标、应用截图、名称、简介描述、企业认证资料等。对于安卓市场，您需要先准备计算机软件著作权，然后可以选择第三方市场或手机厂商市场进行上架。

### 1、准备材料

::: tip 准备材料（安卓）

- **1）账户申请**
  - 在各大应用市场申请开发者账号，目前安卓比较主流的几个市场是华为、应用宝、小米、OPPO、VIVO 市场，有些朋友也会准备魅族的市场，根据自己需求来申请。
- **2）企业资质**
  - 企业开发者企业认证资料，公司营业执照副本、法人身份证照片、开户行许可证、ICP 备案截图。
- **3）上架物料**
  - apk 包
  - icon 图标
  - 应用截图
  - app 名称
  - 20 字以内一句话简介
  - 200 字左右的介绍描述
- **4）其他资料**
  - 留联系方式：邮箱、手机号
- **5）上架资质**
  - 软著权
  - 软件著作权证书或者电子版权证书、icp 备案截图，部分市场部分类别可以不用软著，在后台下载承诺函，<span style="color:#f00;font-weight:bold;text-decoration:underline;">特殊行业需要对应的资质</span>。
  - 其他资质：《ICP 许可证》《网络文化经营许可证》《信息网络传播视听节目许可证》《广播电视节目制作经营许可证》《游戏运营备案》《EDI 许可证》《金融许可证》等

:::

::: danger 注意
绝大部分的安卓应用商店上架的基本条件就是软著。
:::

### 2、上架流程

> 周期 3 天左右。

::: tip 上架流程

1. 注册开发者账号

2. 提交应用信息

3. 提交应用包

4. 应用审核

5. 上线发布

:::

## 二、IOS

> 在苹果 Store 上架时，需要申请苹果开发者账号，并进行 APP ID 创建和证书申请，配置相关信息后打包上传。最后提交 APP 并等待审核成功。

### 1、准备材料

::: tip IOS 准备材料

1. 应用名称：和 APP 桌面显示名称一致。

2. 应用描述：描述展现在 appstore 上，尽量多写点，最低 30 字+。

3. 关键字（SEO 用）：appstore 市场上搜索的相关关键词，最多 10 个。

4. 支持的网页：官网网站（与 APP 同一主体）。

5. 市场销售链接：免费的 APP 跳过此项。

6. iPhone 应用截图（3-5 张）：尺寸大小 1242\*2208

   iPhone XR 应用截图（3-5 张）：尺寸大小 1242\*2688

   上架 appstore 需要准备的材料（全部为 JPG）===必须是应用截图，不得是启动页

7. 联系人姓名：您的姓名。

8. 联系人邮箱：您的常用邮件。

9. 联系人电话：您的电话，座机或者手机。

10. demo 账号：演示帐号，演示最好是新建的空白帐号，如果 APP 不需要登录，那么请跳过此项。

11. demo 密码：演示帐号密码。

12. 隐私网址：网址打开的内容为你 app 的隐私政策。

13. 苹果开发者账号：注册并付费成功的账号。

14. ipa 安装包：用开发者账号制作证书并添加到安装包里面的新包。

注：软著、网站、电信增值业务许可证需提前办理。
:::

### 2、上架流程

> 周期一个礼拜左右。

::: tip IOS 上架流程（简化）
一个包、 两个网址、 三个证书

- 一个包：iPA 包，上架用的。
- 两个网址：
  - `https://itunesconnect.apple.com`
  - `https://developer.apple.com/`
- 三个证书：
  - 签名证书.(MAC 电脑钥匙串生成)
  - 制作发布证书.(签名证书+开发者账号)
  - 制作 PP 配置文件.(发布证书+App ID)

:::

::: tip IOS 上架流程

1. 申请苹果开发者账号

2. 创建 APP ID 以及申请证书

3. 在开发者中心的 iTunes Connect 中配置 App 相关信息

4. 打包上传 IOS APP

5. 在 ios store 提交 APP，上线成功

:::
