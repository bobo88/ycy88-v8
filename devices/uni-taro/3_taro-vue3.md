# Taro VUE3 项目

## 一、初始化项目

> 使用命令创建模板项目：

```bash
$ taro init taro-vue3
```

![An image](/images/devices/taro-ui-1.png)

## 二、运行项目

> 创建 `VUE3` 项目，并通过 `yarn dev:weapp` 运行命令（需要下载配置 `微信开发者工具`并引入当前项目）。

![An image](/images/devices/taro-ui-2.png)
